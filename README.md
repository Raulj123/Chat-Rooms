## Install

#### Create and activate virtual environment

`python -m venv venv`

`source venv/bin/activate`

### Install dependencies

`pip install -r requirements.txt`

### To Run

`uvicorn main:app --reload`

open up a new terminal

`cd frontend`

`npm install`

`npm run dev`
